# MLX-Audio TTS Integration

MLX-Audio is a Swift-compatible TTS library built on Apple's MLX framework, optimized for Apple Silicon. It offers high-quality voice synthesis with the Kokoro model (82M parameters), voice cloning capabilities, and 40% faster performance than traditional frameworks. 

Installation: `pip install mlx-audio` or Swift Package Manager integration.

Complements existing WhisperKit STT to create bidirectional voice assistant functionality.
